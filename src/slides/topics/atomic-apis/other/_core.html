CSS Custom Properties

  • Enable declarative overrides.
  • Provide flexibility through variable redefinition.
  • Serve as the building blocks of your design tokens.

CSS Layers

  • Establish predictable priorities.
  • Define distinct layers for base styles, components, and overrides.
  • Ensure a structured cascading order.

Organizing Your Design System as a CSS API

  • Clear and semantic naming conventions.
  • Composable and overridable components.
  • Easy customization for external consumers.

Live Snippet Example


@layer base, components, overrides;

:root {
  --font-size-base: 16px;
  --color-bg-default: #ffffff;
  /* Other tokens... */
}

This snippet demonstrates how to layer your CSS and define tokens for a clear, composable API.
